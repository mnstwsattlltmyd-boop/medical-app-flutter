## 🚀 الخطوات التفصيلية:

---

## **الخطوة 1️⃣: إنشاء حساب GitHub (إذا لم يكن لديك)**

### أ) انتقل إلى موقع GitHub:
```
https://github.com
```

### ب) اضغط "Sign up" (التسجيل):
- أدخل بريدك الإلكتروني
- اختر اسم مستخدم (مثل: `medical-app-user`)
- اختر كلمة مرور قوية
- اضغط "Create account"

### ج) تحقق من بريدك الإلكتروني:
- GitHub سيرسل لك رسالة تحقق
- اضغط على الرابط في الرسالة

✅ **تم! لديك حساب GitHub الآن**

---

## **الخطوة 2️⃣: رفع مشروعك على GitHub**

### أ) على جهازك، افتح Command Prompt/Terminal:

```bash
# انتقل إلى مجلد المشروع
cd flutter_medical_app
```

### ب) تهيئة Git:

```bash
git init
git config user.name "Your Name"
git config user.email "your@email.com"
```

### ج) إضافة الملفات:

```bash
git add .
git commit -m "Initial commit - Medical App"
```

### د) إنشاء مستودع على GitHub:

1. انتقل إلى: https://github.com/new
2. اسم المستودع: `medical-app-flutter`
3. الوصف: `Medical Care App with Flutter`
4. اختر "Public" (عام)
5. اضغط "Create repository"

### هـ) رفع المشروع:

```bash
git remote add origin https://github.com/YOUR_USERNAME/medical-app-flutter.git
git branch -M main
git push -u origin main
```

⚠️ **ملاحظة:** استبدل `YOUR_USERNAME` باسم مستخدمك على GitHub

✅ **تم! المشروع الآن على GitHub**

## **الخطوة 8️⃣: نقل الملف إلى الهاتف**

### الطريقة 1️⃣: عبر البريد الإلكتروني

1. افتح البريد الإلكتروني على جهازك
2. أنشئ رسالة جديدة لنفسك
3. أرفق ملف `app-release.apk`
4. أرسل الرسالة
5. افتح البريد على الهاتف وحمّل الملف

### الطريقة 2️⃣: عبر Google Drive

1. انتقل إلى: https://drive.google.com
2. اضغط "New" > "File upload"
3. اختر ملف `app-release.apk`
4. انتظر انتهاء الرفع
5. على الهاتف، افتح Google Drive وحمّل الملف

### الطريقة 3️⃣: عبر كابل USB

1. وصّل الهاتف بالكمبيوتر عبر USB
2. انسخ الملف إلى مجلد على الهاتف
3. افتح مدير الملفات على الهاتف

✅ **تم! الملف على الهاتف الآن**

---

## **الخطوة 9️⃣: تثبيت التطبيق على الهاتف**

### أ) السماح بالتثبيت من مصادر غير معروفة:

1. افتح **Settings** (الإعدادات)
2. اذهب إلى **Apps** أو **Applications**
3. ابحث عن **Special app access** أو **Install unknown apps**
4. اختر **Chrome** أو **Files** (حسب ما استخدمته)
5. فعّل **"Allow installing unknown apps"** ✅

### ب) فتح الملف:

1. افتح **Files** (مدير الملفات)
2. اذهب إلى **Downloads**
3. ابحث عن `app-release.apk`
4. اضغط عليه

### ج) التثبيت:

1. ستظهر نافذة التثبيت
2. اضغط **"Install"**
3. انتظر قليلاً (30 ثانية تقريباً)
4. اضغط **"Open"** لتشغيل التطبيق

✅ **تم! التطبيق مثبّت على هاتفك!**

## 🆘 مشاكل شائعة وحلولها:

### ❌ مشكلة: "Build failed"

**الحل:**
1. تأكد من أن جميع الملفات موجودة
2. تحقق من أن `pubspec.yaml` صحيح
3. جرّب البناء مرة أخرى

---

### ❌ مشكلة: "GitHub authorization failed"

**الحل:**
1. تأكد من تسجيل الدخول إلى GitHub
2. أعطِ Codemagic إذن الوصول
3. جرّب مرة أخرى

---

### ❌ مشكلة: "Installation failed on device"

**الحل:**
1. تأكد من تفعيل "Unknown sources"
2. احذف النسخة القديمة من التطبيق
3. جرّب التثبيت مرة أخرى

---

### ❌ مشكلة: "File not found"

**الحل:**
1. تأكد من انتهاء البناء
2. تحقق من وجود الملف في Artifacts
3. جرّب البناء مرة أخرى
